## 🎯 **O Problema**

Existem **300+ frameworks** de consultoria validados.  
SWOT, Canvas, BPMN, Lean, OKR, 5 Porquês...

**O paradoxo:** Mais opções = paralisia decisória (Barry Schwartz)

EJs sabem que frameworks funcionam, mas:
- ❌ Dá trabalho aplicar manualmente
- ❌ Não sabem qual escolher
- ❌ Não têm tempo pra fazer "do jeito consultoria"

---

## 💡 **A Solução**

**Framework = Processo Validado**

Agentes de IA executam frameworks pra você.

**Resultado:**  
- ✅ Mesma qualidade de consultoria enterprise  
- ✅ Velocidade de startup (10 dias → 30 minutos)  
- ✅ Você foca em insights, não formatação

---

## 🗺️ **As Duas Jornadas**

### **IDEIA → PROCESSO**
```
ETAPA 1: Validar Impacto (ICE/RICE Score)
   ↓
ETAPA 2: Mapear Jornada (Customer Journey)
   ↓
ETAPA 3: Estruturar Replicável (SOP/Playbook)
```

### **DOR → PROCESSO**
```
ETAPA 1: Identificar Causa Raiz (5 Porquês/Ishikawa)
   ↓
ETAPA 2: Mapear Processo Atual (BPMN As-Is)
   ↓
ETAPA 3: Redesenhar Processo (BPMN To-Be)
```

## 📖 **Caso Real: ConsultAI**

Este repositório é baseado no **ConsultAI** — plataforma multi-agente que uso na BIP.

**Antes:** Diagnóstico de maturidade levava **3-4 semanas**  
**Depois:** Mesmo diagnóstico em **48 horas**  

**Cliente:** C.Vale Cooperativa Agroindustrial (R$ 20bi)  
**Stack:** CrewAI + FastAPI + Supabase + OpenRouter  
**Status:** Produção desde 2024

**Orquestração:**
- CrewAI (multi-agente)
- LangChain (alternativa)

**Backend:**
- FastAPI (Python)
- Supabase (dados + vector)
