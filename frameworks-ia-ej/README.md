## 💡 **A Solução**

**Framework = Processo Validado**

Agentes de IA executam frameworks pra você.

**Resultado:**  
- ✅ Mesma qualidade de consultoria enterprise  
- ✅ Velocidade de startup (10 dias → 30 minutos)  
- ✅ Você foca em insights, não formatação

---

## 🗺️ **As Duas Jornadas**

### **IDEIA → PROCESSO**
```
ETAPA 1: Validar Impacto (ICE/RICE Score)
   ↓
ETAPA 2: Mapear Jornada (Customer Journey)
   ↓
ETAPA 3: Estruturar Replicável (SOP/Playbook)
```

### **DOR → PROCESSO**
```
ETAPA 1: Identificar Causa Raiz (5 Porquês/Ishikawa)
   ↓
ETAPA 2: Mapear Processo Atual (BPMN As-Is)
   ↓
ETAPA 3: Redesenhar Processo (BPMN To-Be)
```

## 🔥 **Frameworks Incluídos**

| Etapa | Framework | Quando usar |
|-------|-----------|-------------|
| **Diagnóstico** | ICE Score | Priorizar ideias rapidamente |
| | RICE Score | Validar investimento grande |
| | 5 Porquês | Causa raiz de problema simples |
| | Ishikawa | Problema com múltiplas causas |
| **Mapeamento** | Customer Journey | Entender experiência do usuário |
| | BPMN As-Is | Mapear processo atual |
| | Value Stream | Identificar desperdícios (Lean) |
| **Estruturação** | BPMN To-Be | Redesenhar processo otimizado |
| | SOP | Criar procedimento padrão |
| | Playbook | Documentar metodologia completa |
